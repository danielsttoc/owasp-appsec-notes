# OWASP AppSec Notes

A practical application security notes repository documenting my learning from OWASP Top 10 labs, SecureFlag exercises and hands-on web security practice.

This repo is designed to show how I understand common web application vulnerabilities, how they are exploited in controlled lab environments and how they can be prevented in real systems.

## Focus Areas

- OWASP Top 10
- SQL injection
- Cross-site scripting
- Broken access control
- Broken authentication
- Insecure deserialization
- Security misconfiguration
- Sensitive information disclosure
- Insufficient logging and monitoring
- Secure coding principles

## Why I Made This Repo

I am currently studying Computing with a focus on Networks, Cyber Security and Forensics at Edge Hill University. Alongside university, I have been building hands-on experience through my homelab, SecureFlag labs, Docker security practice and SIEM monitoring work.

This repository helps me document what I have learned in a clear, professional way so I can revise the topics, explain them in interviews and show practical evidence of my AppSec learning.
